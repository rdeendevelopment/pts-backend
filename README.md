## CONFIGURATION
Please create config.yaml file in main directory and add required variables mentioned in config-live.yaml

## Database
Create a MongoDB database and add `mongodb.uri` and `mongodb.db` in `config.yaml`.

## Libraries and Dependencire
Install libraries and dependencire with `npm install`  command

## Development server
* Run `node server.js` for a dev server. Navigate to `http://localhost:3000/`. 
    Or
* Run with `nodemon` The app will automatically reload if you change any of the source files.
